# Mobile Gemini AI Agent (Python Terminal)

Ένας αυτόνομος AI Agent που αναπτύχθηκε και τρέχει εξ ολοκλήρου σε περιβάλλον Android (μέσω Pydroid 3), χρησιμοποιώντας τη νέα επίσημη βιβλιοθήκη `google-genai`.

## 🚀 Χαρακτηριστικά
- **Modern SDK**: Πλήρης χρήση του ανανεωμένου `google-genai` client της Google.
- **Optimized Model**: Χρήση του `gemini-2.5-flash` για αστραπιαίες απαντήσεις με χαμηλή κατανάλωση πόρων.
- **Zero-GUI Architecture**: Σχεδιασμένο να τρέχει απευθείας στο Terminal για 100% σταθερότητα σε mobile περιβάλλοντα.

## 🛠️ Τεχνολογίες
- Python 3
- Google GenAI SDK (Gemini API)

## 📦 Πώς να το τρέξετε
1. Κάντε clone το repository.
2. Εγκαταστήστε τη βιβλιοθήκη: `pip install google-genai`
3. Αποκτήστε ένα δωρεάν API Key από το Google AI Studio.
4. Προσθέστε το κλειδί σας στη γραμμή 8 του αρχείου `main.py`.
5. Τρέξτε την εφαρμογή: `python main.py`
